# 📌 Project Overview

This project demonstrates a complete Machine Learning Regression workflow for predicting house prices using real estate data. The project covers data preprocessing, feature engineering, multiple regression algorithms, regularization techniques, and gradient descent optimization methods.

The objective is to understand how different regression models perform on the same dataset and compare their prediction accuracy.

# ⚡ Gradient Descent Algorithms

This project includes implementation of Gradient Descent techniques from scratch for deeper understanding of optimization in Machine Learning.

---

## 🔹 Batch Gradient Descent

Uses the entire dataset to update model parameters.

### Advantages

* Stable convergence
* Accurate gradient estimation

### Disadvantages

* Slower for large datasets

---

## 🔹 Mini-Batch Gradient Descent

Uses small batches of data during each update.

### Advantages

* Faster than Batch GD
* Efficient memory usage

### Disadvantages

* Slightly noisy updates

---

## 🔹 Stochastic Gradient Descent (SGD)

Updates parameters using one training example at a time.

### Advantages

* Very fast learning
* Efficient for large datasets

### Disadvantages

* More fluctuations during training

# 📈 Model Evaluation Metrics

The following evaluation metrics were used:

### MAE (Mean Absolute Error)

Measures average prediction error.

### MSE (Mean Squared Error)

Penalizes larger prediction errors.

### RMSE (Root Mean Squared Error)

Provides error in original units.

### R² Score

Measures model goodness-of-fit.
